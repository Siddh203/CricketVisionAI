CricketVision AI

A computer vision cricket coaching app that analyzes batting videos and classifies cricket shots using pose estimation and machine learning.

Current Features:
- Reads cricket batting videos
- Detects body pose using MediaPipe
- Draws pose landmarks on each frame
- Extracts keypoints into CSV format for training
- Supports early dataset creation for cover drive, pull shot, and sweep

Tech Stack:
Python, OpenCV, MediaPipe, Pandas, Scikit-learn
